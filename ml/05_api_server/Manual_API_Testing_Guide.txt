=========================================================================================================
DEFENSE Q&A: HOW TO MANUALLY TEST THE API (WITHOUT THE FRONTEND)
=========================================================================================================

If a professor asks: "Can you prove the backend API works independently of the website?" or if you just want to show off your deep technical understanding, use the two methods below.

Both of these methods prove that the Machine Learning model is fully operational and actively listening on port 8000, even if the React website is turned off.

---------------------------------------------------------------------------------------------------------
METHOD 1: THE "HACKER" WAY (Terminal cURL Request)
---------------------------------------------------------------------------------------------------------
This proves you understand raw HTTP protocols and command-line interfaces.

1. Ensure the FastAPI server is running in the background.
2. Open a new Terminal window.
3. Copy and paste this exact command and hit Enter:

curl -X POST "http://localhost:8000/predict" \
-H "Content-Type: application/json" \
-d '{"state": "Maharashtra", "district": "nashik", "market": "Lasalgaon", "commodity": "Onion", "date": "2024-06-15"}'

4. The server will intercept this raw JSON, run the math, and instantly spit this exact response into your terminal:

{"predicted_price":1987.7,"status":"success"}


---------------------------------------------------------------------------------------------------------
METHOD 2: THE "SENIOR ENGINEER" WAY (Interactive Swagger UI)
---------------------------------------------------------------------------------------------------------
Because this server was built using FastAPI, it automatically generates an interactive, industry-standard documentation page. This proves your API is professionally built.

1. Ensure the FastAPI server is running.
2. Open any web browser (Chrome, Safari, etc.).
3. Go to this exact URL: 
   http://localhost:8000/docs

4. Once the beautiful UI loads:
   - Click on the green "POST /predict" route.
   - Click the "Try it out" button in the top right corner.
   - You will see an editable JSON text box. You can change "Lasalgaon" to another market, or change the date.
   - Click the big blue "Execute" button.
   - Scroll down to see the AI's predicted price generated in real-time.
